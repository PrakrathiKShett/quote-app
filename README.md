# 📱 DQuotes – Quote of the Day App

DQuotes is a simple and attractive web-based application that displays motivational and inspiring quotes to users. The app is designed with a modern UI and provides a smooth user experience.

---

## 🚀 Features

- 🎯 Displays random motivational quotes
- 🔄 Generate a new quote with one click
- 📤 Share quotes easily (copy to clipboard)
- 🎨 Modern gradient UI design
- 📱 Mobile-like layout

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript

## ⚙️ How It Works

- Quotes are stored in a JavaScript array
- A random quote is selected using JavaScript logic
- The selected quote is displayed dynamically on the screen
- The share button copies the quote for easy sharing
